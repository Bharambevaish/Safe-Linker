# Safe-Linker 🛡️
A lightweight URL Safety Checker designed to identify potentially malicious links before you click them.

## 🚀 Features
* **Real-time Analysis:** Checks URLs against known blacklists.
* **Safety Scoring:** Provides a risk level for submitted links.
* **Clean UI:** Simple interface for quick link verification.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, JavaScript (or Angular)
* **Backend:** Java / Spring Boot (or Python/Node.js)
* **APIs:** Google Safe Browsing / VirusTotal (suggested)

## 📋 How to Run
1. Clone the repo: `git clone https://github.com/Bharambevaish/Safe-Linker.git`
2. Install dependencies: `npm install` (or your specific setup)
3. Run the application: `npm start`
